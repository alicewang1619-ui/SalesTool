# 页面文档 · 检索结果

- **文件**：mvp_检索_检索结果.html
- **阶段/模块**：MVP / 语义检索（FR-3）
- **路由**：`/search?q=...`

## 1. 页面目标
展示语义检索结果——按相关度排序的知识卡片，命中片段高亮，让用户快速定位并跳转。

## 2. 用户与权限
单用户本地自用。

## 3. 布局结构
顶部搜索框（带当前 query）+ 结果头（命中数 + query + 排序说明）+ 引导改用问答的提示条 + 结果卡片流。

## 4. 核心组件
- 结果卡片：标题 / 命中片段（高亮）/ 相关度徽标 / 标签 / 来源
- 「改用问答」提示条 → 问答页

## 5. 数据来源
对 query 做 embedding → 本地向量库检索 Top-K → 返回知识 + 相关度分数 + 命中片段。

## 6. 状态设计
- **搜索中**：结果区骨架/加载。
- **无结果**：空态「没找到相关知识」+ 建议换个说法 / 去问答 / 去录入。
- **错误**：embedding/向量库异常时提示重试。

## 7. 交互
- 点卡片 → 知识详情
- 改写 query 重新搜
- 点提示条 → 问答页

## 8. 验收（关联 AC）
- **AC-3**：口语化输入返回语义相关结果，相关项排在前；命中片段高亮。
- 无结果有空态引导。

## 9. 跳转关系
→ 知识详情 / 问答 / 录入。
