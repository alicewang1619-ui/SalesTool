## 4. Mock 策略

### 4.1 必须 Mock 的外部依赖
| 依赖 | Mock 方式 | 理由 | 关联用例 |
| :--- | :--- | :--- | :--- |
| 检索接口 | 返回结果集/空/加载延迟 | 驱动结果/空态/加载态 | C-ST-01/02/03/05 |
| 标签查询接口 | 返回标签集/空 | 驱动标签云/空态 | C-ST-06/07/08 |

### 4.2 禁止 Mock 的内部单元
| 单元 | 理由 |
| :--- | :--- |
| 命中高亮渲染 | 被测对象本身 |
| 标签聚合交互 | 被测对象本身 |

> Mock 检索结果须含 command 片段与相关度字段，否则无法验证高亮与排序展示。
