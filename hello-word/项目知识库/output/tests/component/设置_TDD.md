## 4. Mock 策略

### 4.1 必须 Mock 的外部依赖
| 依赖 | Mock 方式 | 理由 | 关联用例 |
| :--- | :--- | :--- | :--- |
| 模型连接探测 | 返回已连接/未就绪 | 驱动状态徽标 | C-SET-02 |
| 导出/恢复接口 | 返回成功 | 隔离持久层 | C-SET-04/05 |

### 4.2 禁止 Mock 的内部单元
| 单元 | 理由 |
| :--- | :--- |
| 模型切换交互 | 被测对象本身 |
| 清空二次确认交互 | 被测对象本身 |

> 不得 mock 掉 Key 输入框的 type 属性断言——脱敏是真实 DOM 行为。
