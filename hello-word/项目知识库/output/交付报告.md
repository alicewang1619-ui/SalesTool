### 7.1 浏览器一键剪藏
- **后端**：`POST /api/ingest/clip { url, html }` —— 接收浏览器**已渲染**页面的 outerHTML，复用 `extractArticle`（含公众号适配）提取正文、清洗脚本、入库（`source_type=link`、回填 `source_url`）。天然绕过反爬、支持需登录页面。
- **扩展**：`app/extension/`（Chrome Manifest V3）—— 弹窗一键剪藏当前页，后端地址可配，成功给「打开查看」链接。权限仅 `activeTab/scripting/storage` + 本机 host，数据不出本机。加载方式见 `app/extension/README.md`。
- **实测**：公众号样例 → `viaWeixin=true`、og:title 标题、正文提取、`<script>` 清洗、`source_url` 落库。

### 7.2 公众号抓取增强（`extract.ts`，clip 与直抓 link 共用）
- og:title 标题兜底链（`#activity-name → og:title → h1 → title`）
- `<section>` 段落换行、剔除赞赏/关注/二维码等噪音节点
- 识别反爬/验证拦截页（"环境异常/去验证"等）→ 抛明确错误，驱动前端回退「剪藏扩展」或「粘贴正文」
- **实测**：噪音"赞赏"已剔除、脚本已清洗。

### 7.3 关系图谱可视化
- **后端**：`services/graph.ts` + `GET /api/graph?minRelevance=&limit=` —— 以每条知识的向量**质心**两两余弦构边（≥阈值），无向去重、节点上限防 O(n²) 爆炸；返回 `{nodes[{id,title,tags,degree}], edges[{source,target,weight}]}`。
- **前端**：`/graph` 页（左侧导航「🕸️ 图谱」）—— **零依赖**力导向 SVG 布局，节点大小=关联度、连线深浅=语义相近度、可调关联强度阈值、点击节点进详情、空态引导。无设计原型，按 DESIGN.md 令牌实现（已记纪要）。
- **实测**：返回真实关系网（4 节点 6 边，含用户自有知识）。
