# 页面文档 · 设置

- **文件**：mvp_设置_设置.html
- **阶段/模块**：MVP / 大模型底座 + 运维（FR-7、FR-8）
- **路由**：`/settings`

## 1. 页面目标
配置大模型（本地/云端）、数据导出备份、回收站管理。

## 2. 用户与权限
单用户本地自用。

## 3. 布局结构
三张卡片：① 大模型（本地 Ollama / 云端 API 二选一 + 隐私提示）② 数据与备份（导出/导入 + 占用统计）③ 回收站（已删知识恢复/清空）。

## 4. 核心组件
- 大模型单选项（本地/云端，含连接状态徽标、API Key 输入）
- 云端「服务商」预设下拉（Google AI Studio(Gemini) / DeepSeek / 通义 / 智谱 / 自定义），选中自动回填 baseUrl 与建议模型名
- 隐私提示条
- 导出/导入按钮、数据统计
- 回收站列表（恢复/清空）

## 5. 数据来源/写入
模型配置（本地/云端、API Key、模型名）；数据统计；回收站（软删 Knowledge）。

## 6. 状态设计
- **本地模型未就绪**：徽标显示未连接 + 引导启用。
- **云端 Key 失效**：明确报错。
- **回收站空**：空态提示。

## 7. 交互
- 切换模型、填 API Key、测试连接
- 导出/导入备份
- 回收站恢复/清空（清空二次确认）

## 8. 验收（关联 AC）
- **AC-7**：可在本地/云端间切换且均可用。
- **AC-8**：数据全本地，可一键导出备份；删除可在回收站恢复。

## 9. 跳转关系
被「问答（模型不可用）」「知识删除（回收站）」引用。
