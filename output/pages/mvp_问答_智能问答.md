# 页面文档 · 智能问答（RAG）

- **文件**：mvp_问答_智能问答.html
- **阶段/模块**：MVP / 库内问答（FR-6）
- **路由**：`/ask`

## 1. 页面目标
基于个人知识库问答——用户提问，AI 检索相关知识生成答案并标注可点击来源。

## 2. 用户与权限
单用户本地自用。

## 3. 布局结构
对话式：顶部标题 + 当前模型标识 + 消息流（用户问 / AI 答）+ 底部输入框。AI 答案含正文 + 行内引用角标 + 底部来源列表。答案正文支持 Markdown 与 LaTeX 数学公式（行内 $…$ / 块级 $$…$$，与知识详情页同一渲染）。

## 4. 核心组件
- 用户/AI 消息气泡
- 答案卡片（正文 + 行内 cite 角标 + 来源列表）
- 来源条目（编号 + 标题，可跳转）
- 输入框 + 发送
- 思考中状态、模型标识

## 5. 数据来源
问题 → 检索相关知识块（Top-K）→ 拼上下文 → 大模型生成答案 + 来源 knowledge_id。

## 6. 状态设计
- **思考中**：检索/生成中的动画提示。
- **无相关知识**：提示「库里没有相关内容，换个问法或先去录入」。
- **模型不可用**：本地 Ollama 未启动 → 提示去设置启用/切云端。

## 7. 交互
- 提问 → 流式答案
- 点行内角标/来源 → 知识详情
- 切换模型在设置页

## 8. 验收（关联 AC）
- **AC-6**：返回答案 + 可点击跳转的来源知识列表。
- **AC-7**：仅本地大模型时可用，切云端同样可用。

## 9. 跳转关系
→ 知识详情（点来源）/ 设置（模型不可用）。
