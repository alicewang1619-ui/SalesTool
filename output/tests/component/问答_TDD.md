## 4. Mock 策略

### 4.1 必须 Mock 的外部依赖
| 依赖 | Mock 方式 | 理由 | 关联用例 |
| :--- | :--- | :--- | :--- |
| 问答接口 | 返回带来源答案/无相关/模型不可用错误 | 驱动各分支 | C-QA-02/04/05 |

| 单元 | 理由 |
| :--- | :--- |
| 消息流渲染 | 被测对象本身 |
| 来源角标跳转 | 被测对象本身 |
