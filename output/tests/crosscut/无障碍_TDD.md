## 4. Mock 策略

### 4.1 必须 Mock 的外部依赖
| 依赖 | Mock 方式 | 理由 | 关联用例 |
| :--- | :--- | :--- | :--- |
| 无 | 无障碍检查针对真实 DOM | 必须测真实渲染结构 | 全部 |

### 4.2 禁止 Mock 的内部单元
| 单元 | 理由 |
| :--- | :--- |
| 真实 DOM 结构与 ARIA 属性 | 无障碍正是验证真实可访问性 |

> 用 axe-core 扫真实渲染 DOM，不可用 mock 组件替代。
