## 4. Mock 策略

### 4.1 必须 Mock 的外部依赖
| 依赖 | Mock 方式 | 理由 | 关联用例 |
| :--- | :--- | :--- | :--- |
| embedding 模型调用 | 返回固定维度确定向量 | 真实模型不确定、慢 | U-SR-02/03/05 |
| 向量库检索 | 返回可控相似度结果集 | 隔离存储层、专注排序逻辑 | U-SR-05/09 |

### 4.2 禁止 Mock 的内部单元
| 单元 | 理由 |
| :--- | :--- |
| 分块算法 | 被测对象本身 |
| Top-K 排序与归一化 | 被测对象本身，mock 等于没测 |
| RAG 上下文拼装与来源收集 | 被测对象本身 |

> Mock 向量必须维度一致、可区分相似度梯度，否则无法验证排序正确性。
