# 我的账号与邮箱

## 1. 文档元信息

| 字段 | 内容 |
| :--- | :--- |
| 所属产品 | 智能销售接待与客户洞察 Agent |
| 所属模块 | 账号权限 |
| 适用端 | Web 后台 |
| 页面路由 | `/admin/me` |
| 页面类型 | 表单 / 个人设置 |
| 所属阶段 | 阶段1(MVP) |
| 上游来源 | 左侧导航“我的”、再营销邮件发送缺少邮箱时的引导 |
| 下游去向 | 再营销待办、设置管理 |

## 2. 页面目标

管理员、运营、销售都能自助维护自己的账号资料、密码和邮箱配置；销售配置邮箱后，可在再营销模块给自己负责国家或本人分配客户发送邮件。

## 3. 页面结构

1. 账号资料：姓名、登录邮箱、角色、数据范围，只允许修改本人姓名和联系邮箱展示名。
2. 密码修改：旧密码、新密码、确认新密码，保存后写审计并提示重新登录或继续使用当前会话。
3. 邮箱配置：发件邮箱、SMTP/邮箱服务类型、显示名称、测试发送按钮、配置状态。
4. 权限提示：管理员/运营可对所有潜在客户发邮件；销售仅可对本人客户范围发邮件。

## 4. 交互规则

- 邮箱未配置时，再营销草稿详情的“人工确认发送”按钮禁用，并展示“去我的配置邮箱”。
- 销售修改邮箱只影响本人发件身份，不影响系统主邮箱。
- 管理员/运营的个人邮箱可作为发件邮箱；系统主邮箱仍在设置页全局配置中维护。
- 密码保存失败必须保留表单输入并展示错误原因；成功必须写审计。

## 5. 权限

| 角色 | 查看 | 修改账号资料 | 修改密码 | 配置个人邮箱 |
|---|---|---|---|---|
| 管理员 | 是 | 本人 | 本人 | 本人 |
| 运营 | 是 | 本人 | 本人 | 本人 |
| 销售 | 是 | 本人 | 本人 | 本人 |
