## 2. 测试策略
- P0：登录、线索池、客户详情、设置管理、账号权限、再营销草稿详情的关键控件必须渲染并按权限变化。
- P1：列表筛选、Banner 管理入口、提示词/附件、下一步动作展示和保存态必须稳定。
- 不测：接口真实返回和数据库写入，由集成契约层负责。
