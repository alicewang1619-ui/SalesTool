# TDD 测试设计文档：安全鉴权

## 1. 单元元信息
| 字段 | 内容 |
| :--- | :--- |
| 测试层 | 安全鉴权 |
| 单元短码 | SEC |
| 对应上游产物 | PRD 7.4、AC-15/16/17/19；技术方案.md 安全、权限、审计 |
| 覆盖范围 | 所有受保护页面、接口、导出、配置、反馈链接和上传入口 |

## 2. 测试策略
- P0：登录、会话、RBAC、销售数据隔离、反馈链接、配置权限、上传安全零失败。
- P1：审计、脱敏、CSRF/XSS/注入和退订/不再联系标记完整。
- 不测：渗透测试报告格式，由后续安全专项补充。

| ID | 用例名称 | 优先级 | Given | When | Then | 预期失败原因 | 来源 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| S-SEC-01 | 未登录访问后台跳转登录 | P0 | 无 session | 访问 `/workspace` | 返回登录页或 401 | 未实现时匿名用户可访问后台 | 技术方案认证 |
| S-SEC-02 | 销售水平越权访问他人线索返回 403 | P0 | 销售 A token | GET 销售 B 线索详情 | 返回 403 并写审计 | 未实现时客户信息泄露 | PRD AC-19 |

| S-SEC-04 | 篡改 JWT 不能伪造管理员 | P0 | 销售 token 修改 role=admin | 调用管理员接口 | token 校验失败 | 未实现时可垂直越权 | 技术方案 JWT/RBAC |

| S-SEC-12 | 再营销发送必须带退订/不再联系标记能力 | P1 | 确认发送再营销邮件 | 生成发送请求 | 包含退订或不再联系标记，发送审计完整 | 未实现时邮件合规风险不可控 | PRD 7.4 |
