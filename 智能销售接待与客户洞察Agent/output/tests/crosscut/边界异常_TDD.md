# TDD 测试设计文档：边界异常

## 1. 单元元信息
| 字段 | 内容 |
| :--- | :--- |
| 测试层 | 边界异常 |
| 单元短码 | EDGE |
| 对应上游产物 | PRD 边界条件；技术方案 风险；pages/*.md 状态与校验 |
| 覆盖范围 | 导入、表单、状态机、异步任务、上传、网络异常、并发 |

## 2. 测试策略
- P0：不能因为空值、重复提交、非法状态、任务失败造成数据错乱或越权。
- P1：错误提示、重试、保留旧状态、失败项可查看。
