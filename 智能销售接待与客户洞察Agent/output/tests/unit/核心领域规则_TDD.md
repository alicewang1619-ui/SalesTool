## 2. 测试策略
- P0：来源归类、销售数据范围、客户背景调查版本、Banner 发布校验必须准确。
- P1：再营销上下文、附件引用、客户池分层和提醒时间规则必须稳定。
- 不测：真实外部 AI、真实邮箱、真实对象存储，由集成契约层 mock 外部边界。

## 6. 2026-07-02 P0 闭环补充
