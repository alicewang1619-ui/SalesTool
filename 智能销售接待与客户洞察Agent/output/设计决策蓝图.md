# 设计决策蓝图（阶段 I 进行中）

## 1 盘子

- 产品定位：内部使用的 B2B 销售增长系统，不做客户登录门户。
- 端形态：PC 管理后台 + 手机销售反馈卡片。
- 响应式策略：PC 管理后台以桌面宽屏为主；手机端只承载销售反馈卡片，避免把复杂后台完整塞进手机。
- 工作台聚合：今日询盘、有效线索、未反馈、待确认再营销草稿、高意向客户。

## 2 信息架构 & 站点地图

- 导航模式：PC 后台使用左侧常驻导航；详情页保留面包屑；销售端反馈卡片不进入完整后台导航。
- 主分区：
  - 工作台：承载每日总览、异常提醒和待办入口。
  - 线索池：管理询盘进入、AI 接待补全、客户画像评分、国家分发和线索详情。
  - 客户池：沉淀高意向客户、有效客户、已报价未回复、资料库客户、已转代理商、无效、撤单/流失和已成交客户。
  - 报表中心：查看日报、月报、季报、年报以及渠道、区域、产品、销售反馈等非金额指标。
  - 再营销待办：查看 AI 生成的再营销建议、邮件草稿、触达理由和人工确认发送队列。
  - 系统配置：维护渠道接入、国家/区域/销售映射、产品字典、评分规则和账号权限。
- 页面调用关系骨架：
  - 工作台 -> 线索池/线索详情/未反馈列表/再营销待办/报表中心。
  - 线索池 -> 线索详情 -> 分发记录 -> 销售反馈卡片。
  - 线索详情 -> 客户详情/客户池记录/再营销草稿。
  - 客户池 -> 客户详情 -> 再营销待办/报表中心。
  - 报表中心 -> 日报/月报/季报/年报 -> 指标明细。
  - 系统配置 -> 渠道、区域销售映射、产品字典、评分规则、账号权限。

## 3 角色 & 权限

- 账号体系：PC 后台使用自建账号密码；销售反馈卡片使用带过期时间的安全链接，第一版不要求销售进入完整后台。
- 角色分层：
  - 管理员：维护系统配置、账号权限、渠道导入配置、国家/区域/销售映射、产品字典、评分规则、AI 接待 playbook 和提醒规则。
  - 营销/运营：使用工作台、线索池、客户池、报表中心和再营销待办；可查看全局线索、调整客户状态、确认再营销草稿、导出报表和催促未反馈。
  - 销售：只查看分配给自己的线索摘要，通过手机反馈卡片提交已联系、已报价、无效、需跟进、已成交等状态和简短客户判断；不进入复杂 PC 后台。
- 同页角色差异：
  - 无权限的一级导航隐藏。
  - 页面内不可执行动作置灰，并给出简短原因。
  - 非负责人打开销售反馈卡片时只读或提示无权限。
  - 管理员配置变更必须记录操作人和时间。

## 4 各功能页面拆解

- 页面拆解原则：第一版按“线索闭环优先”落页，先保证询盘进入、AI 补全、分发、销售反馈、客户池沉淀、报表分析和再营销确认能串起来。
- 工作台：
  - 页面：工作台首页。
  - 骨架：核心指标卡、今日新增线索、有效线索、未反馈提醒、高意向客户、待确认再营销草稿、快捷入口。
  - 状态：无线索时展示历史趋势入口；加载中用骨架屏；接口失败展示重试；无权限时隐藏全局指标。
- 线索池：
  - 页面：线索池列表、线索详情、渠道导入。
  - 骨架：列表支持来源、国家、客户类型、产品、有效性、分发状态、反馈状态筛选；详情聚合会话记录、客户画像、AI 判断理由、背景摘要、评分、分发记录和反馈历史；导入页支持邮箱/官网后台/展会名片批量导入结果。
  - 状态：空列表给出导入入口；导入失败展示失败项；详情加载失败允许返回列表；无导入权限时隐藏导入按钮。
- AI 接待与客户画像：
  - 页面：在“线索详情”内作为会话与画像区，不单独做一级页面；AI 接待 playbook 放入系统配置。
  - 骨架：按代理商/医生两套字段展示身份、国家、应用、需求、预算、历史设备、业务规模、品牌经验等信息，并保留人工修正入口。
  - 状态：缺字段时标记待补全；AI 判断不确定时显示原因和人工修正；无编辑权限时只读。
- 国家分发与销售反馈：
  - 页面：线索详情内分发区、待分配列表、手机销售反馈卡片。
  - 骨架：分发区展示国家/区域/销售匹配结果；待分配列表处理国家缺失或规则缺失；销售反馈卡片展示客户摘要、AI 判断理由、反馈状态、简短客户判断和后续动作。
  - 状态：无匹配销售时进入待分配；链接过期提示重新发送；非负责人打开反馈卡片只读或提示无权限。
- 客户池：
  - 页面：客户池列表、客户详情。
  - 骨架：客户池按高意向、有效持续跟进、已报价未回复、资料库、已转代理商、无效、撤单/流失、已成交分层；客户详情聚合历史线索、反馈、客户状态、再营销记录和态势信号。
  - 状态：无客户时展示来自线索池的进入规则；状态冲突时提示人工确认；无权限时隐藏人工调整入口。
- 报表中心：
  - 页面：报表中心首页、日报/月报/季报/年报视图、指标明细、导出确认。
  - 骨架：展示询盘总量、有效数量、国家分布、渠道质量、代理商/医生占比、产品分类、销售反馈、未及时反馈、撤单、再营销数据、客户池变化；不展示成交金额和报价金额。
  - 状态：当天无线索展示空状态和历史趋势；导出失败可重试；无导出权限时置灰导出按钮。
- 再营销待办：
  - 页面：再营销待办列表、草稿详情/确认发送。
  - 骨架：展示客户、触达理由、推荐时间、AI 邮件草稿、人工确认按钮和发送结果。
  - 状态：无草稿时展示生成规则说明；发送失败保留待发送；无权限时隐藏确认发送。
- 系统配置：
  - 页面：渠道配置、国家/区域/销售映射、产品字典、评分规则、AI 接待 playbook、提醒规则、账号权限。
  - 骨架：配置页以表格/表单为主，所有保存动作记录操作人和时间。
  - 状态：保存失败保持原配置；规则缺失提示影响范围；非管理员隐藏配置入口。
