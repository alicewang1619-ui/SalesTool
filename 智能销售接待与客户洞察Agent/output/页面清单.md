# 页面清单

| 所属阶段 | 功能模块 | 页面名称 | 功能范围描述 |
| :--- | :--- | :--- | :--- |
| 阶段1(MVP) | 账号权限 | 登录页 | PC 后台自建账号登录入口，完成账号密码登录、会话过期回跳和角色识别。 |
| 阶段1(MVP) | 工作台 | 工作台首页 | 聚合今日询盘、有效线索、未反馈、高意向客户和再营销待确认入口。 |
| 阶段1(MVP) | 线索池 | 线索池列表 | 统一展示官网聊天、官网后台、邮箱、展会名片导入线索，支持筛选、排序和批量处理。 |
| 阶段1(MVP) | 线索池 | 线索详情 | 聚合原始询盘、会话、客户画像、评分理由、背景摘要、分发和反馈历史。 |
| 阶段1(MVP) | 线索池 | 渠道导入 | 处理官网后台、邮箱和展会名片的 CSV/Excel 导入、失败项和重复提示。 |
| 阶段1(MVP) | 分发与反馈 | 待分配列表 | 集中处理国家缺失、国家映射缺失或销售负责人缺失的线索。 |
| 阶段1(MVP) | 分发与反馈 | 手机销售反馈卡片 | 销售通过手机在 10-30 秒内查看客户摘要并提交状态、判断和可选备注。 |
| 阶段1(MVP) | 客户池 | 客户池列表 | 按高意向、有效跟进、资料库、已转代理商、无效、撤单/流失等状态沉淀客户。 |
| 阶段1(MVP) | 客户池 | 客户详情 | 聚合客户历史线索、反馈、状态变化、再营销记录和态势信号。 |
| 阶段1(MVP) | 报表中心 | 报表中心首页 | 展示日报、月报、季报、年报入口和非金额核心指标。 |
| 阶段1(MVP) | 报表中心 | 周期报表视图 | 按日/月/季/年查看询盘量、有效量、国家、渠道、产品和销售反馈。 |
| 阶段1(MVP) | 报表中心 | 指标明细 | 查看官网渠道 KPI、未反馈、销售反馈和产品分类等指标明细。 |
| 阶段1(MVP) | 报表中心 | 导出确认弹窗 | 确认导出周期、字段、脱敏策略和不包含成交金额/报价金额。 |
| 阶段1(MVP) | 系统配置 | 国家区域销售映射 | 维护国家、区域和销售负责人的确定性分发规则。 |
| 阶段1(MVP) | 系统配置 | 产品知识库 | 维护 ultrasound 产品类型、型号、应用场景和 AI 接待知识。 |
| 阶段1(MVP) | 异常与权限 | 无权限页 | 当用户访问无权限页面或动作时展示说明和返回入口。 |
| 阶段1(MVP) | 异常与权限 | 销售链接过期页 | 销售反馈安全链接过期、非负责人或无效时提示重新发送。 |
| 阶段2 | 再营销待办 | 再营销待办列表 | 查看 AI 生成的再营销建议、邮件草稿、触达理由和人工确认队列。 |
| 阶段2 | 再营销待办 | 草稿详情确认发送 | 编辑 AI 邮件草稿、确认发送、取消草稿和查看发送结果。 |
| 阶段2 | 客户态势 | 客户态势信号记录 | 记录客户官网公开信息、邮件互动、销售反馈和人工录入的机会信号。 |
| 阶段2 | 系统配置 | 渠道配置 | 维护官网聊天 Webhook、官网后台导入、邮箱同步和展会名片导入配置。 |
| 阶段2 | 系统配置 | 评分规则 | 维护身份、应用、需求、国家和背景可信度等评分规则。 |
| 阶段2 | 系统配置 | AI接待Playbook | 维护 ultrasound 接待脚本、禁止承诺边界和升级销售规则。 |
| 阶段2 | 系统配置 | 提醒规则 | 维护 24 小时销售提醒、48 小时管理者关注和日报生成规则。 |
| 阶段2 | 系统配置 | 账号权限 | 维护管理员、营销/运营、销售账号和 RBAC 权限。 |

## HTML 预览总入口

- 文件：`output/index.html`
- 用途：集中查看设计系统、页面清单、技术方案以及 `output/pages/` 下全部 25 个高保真 HTML 页面。
- 推荐查看顺序：工作台首页 → 线索池列表 → 线索详情 → 手机销售反馈卡片 → 报表中心首页。
