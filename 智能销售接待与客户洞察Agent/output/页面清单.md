# 页面清单

## 2026-07-01 返修后的页面结构

- 一级导航：工作台、线索池、待分配、客户池、再营销、报表、配置、我的。
- 取消一级导航“客户态势”；原客户态势信号记录并入客户详情页，作为“客户态势 / 详细情况”模块。
- 设置页保留 8 个可用入口：销售账号、角色权限、全局 Banner、国家销售映射、产品知识库、客户来源字典、渠道配置、提醒规则；入口必须能进入页面或可操作区域。
- 导入页新增模板下载、字段说明、文件选择反馈、自动分配结果确认。
- 再营销草稿详情新增邮件发送模块：发件人、收件人、主题、正文、参考素材、发送状态。
- 工作台、线索池、待分配、客户池统一展示进入时间并支持今日/昨天/指定日期/全部筛选；工作台指标卡跳转对应明细。
- 新增“我的账号与邮箱”页面，管理员、运营、销售均可访问。

## 2026-07-03 最新页面清单补充：营销技能与知识库独立导航
- 左侧一级导航新增“营销技能”，路由 `/admin/marketing-skills`，模块定位为邮件写手营销 Prompt 技能配置；仅管理员和运营可见。

- 设置管理页 `/admin/settings` 不再承载营销技能和知识库入口；设置中心仅保留账号权限、全局 Banner、线索分发、邮件接口、AI 与模型、配置审计等系统配置。
- 旧 `/admin/settings/product-knowledge` 仅保留兼容历史链接，并应高亮左侧“知识库”。

## 2026-07-03 最新页面清单补充：挖潜客
- 左侧一级导航新增“挖潜客”，路由 `/admin/prospecting`，模块定位为主动拓客方案、候选潜客和人工确认入库；仅管理员和运营可见。
- 挖潜客页面包含拓客方案表单、结构化客户画像、AI 拓客节奏摘要、候选客户列表四块；画像字段按行业/机构类型、岗位/采购角色、公司规模/渠道类型、应用场景、意图关键词、排除条件和渠道配置，不再只用一段文本描述。
- 候选列表必须展示来源渠道、来源说明和来源链接；来源链接用于人工核验候选客户从哪里挖来。
- 候选客户确认入库后跳转或可进入线索池/客户详情，正式记录中保留“从哪里挖来”的来源追溯。
