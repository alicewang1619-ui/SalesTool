# 开发落地说明

## 当前生产化骨架
- `backend/`：FastAPI + SQLAlchemy，已接真实数据库读写；本地默认 SQLite，Docker Compose 使用 Postgres。
- `frontend/`：React + Vite + Ant Design，已接登录、全局 Banner、工作台、线索池、客户详情、设置管理首批路由。
- `docker-compose.yml`：提供 Postgres、Redis、API、前端容器编排入口；worker/scheduler 将在客户背景调查、再营销草稿、邮件发送等长任务落地时补齐。
